<h3 align="center">Desenvolvido por MadeFy by Thiago Sales </h3>

<h3 align="left">Instruções</h3>
<ul>
    <li>Instalar NODEJS 16</li>
    <li>Instalar YARN</li>
</ul>
comando para instalar dependencias 
    yarn

para iniciar o projeto
    yarn start

Build App
    yarn package-mac (para macos)
    yarn package-win (para windows)
    yarn package-linux (para Linux)
